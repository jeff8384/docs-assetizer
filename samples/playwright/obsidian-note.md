## Usage Patterns
1. Use `page.locator()` to create locators for elements on the page.
2. Use `expect(locator).toBeVisible()` to assert that an element is visible.
3. Install with `npm init playwright@latest`.

## Code Examples

```typescript
import { test, expect } from '@playwright/test';

test('basic test', async ({ page }) => {
  await page.goto('https://playwright.dev/');
  await expect(page.getByRole('heading', { name: 'Playwright enables reliable end-to-end testing' })).toBeVisible();
});
```

## Gotchas & Notes
- [WARNING] Locators are strict. Multiple matches = error.
- `page.waitForSelector()` is deprecated — use locator auto-waiting instead.
