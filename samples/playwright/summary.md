## Usage Patterns
1. Use `page.locator()` to create locators for elements on the page.
2. Use `expect(locator).toBeVisible()` to assert that an element is visible.
3. Install Playwright using `npm init playwright@latest` for a full project setup.
4. Configure browsers in `playwright.config.ts` using the `projects` array.

## Gotchas & Notes
1. [WARNING] Locators are strict by default. If more than one element matches a locator, Playwright will throw an error.
2. Note: `page.waitForSelector()` is deprecated in favor of `page.waitForURL()` and locator-based auto-waiting.
